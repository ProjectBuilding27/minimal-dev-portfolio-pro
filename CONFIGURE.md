# Configure Your Portfolio

Welcome! 🎉

This template has been designed so that the majority of your portfolio can be customised from a single file.

---

# 1. Edit Your Information

Open:

```text
data/site.ts
```

Update:

- Your name
- Job title
- Tagline
- Location
- Email address
- Social links

---

# 2. Update Your Hero Section

Inside `data/site.ts` you'll find the `hero` configuration.

Change:

- Badge
- Headline
- Description
- Button text
- Hero code card
- Statistics

---

# 3. Update Your Projects

Inside `projects` replace the example projects with your own.

For each project update:

- Title
- Description
- Technologies
- Preview image
- Live URL
- GitHub URL

---

# 4. Update Your Skills

Replace the skills array with your own technologies.

Example:

```ts
[
  "React",
  "Next.js",
  "TypeScript",
  "Node.js"
]
```

---

# 5. Update Experience

Replace the example companies with your own work history.

---

# 6. Blog Posts

If you're using the included blog, edit the `posts` array.

If you don't need the blog, simply remove the navigation link and page.

---

# 7. Replace Images

Replace the images inside:

```text
public/assets/
```

Recommended size:

```
1600 × 900 px
```

---

# 8. Deploy

Build locally:

```bash
npm run build
```

Deploy using Vercel:

1. Push to GitHub
2. Import into Vercel
3. Click Deploy

Your portfolio will be live in minutes.

---

Need help?

The project is intentionally structured to keep the most frequently edited content inside `data/site.ts`, making it quick and easy to personalise.
