# Support

Thanks for purchasing Minimal Developer Portfolio Pro.

## Before asking for help

Please check:

1. You are using Node.js 20 or newer.
2. You ran `npm install`.
3. You edited `data/site.ts` only for basic content changes.
4. You restarted the dev server after changing config files.

## Common commands

```bash
npm install
npm run dev
npm run build
```

## What support includes

- Installation help
- Deployment guidance
- Clarification on template structure
- Bug fixes for the original template

## What support does not include

- Custom feature development
- Full redesigns
- Third-party service setup
- Debugging heavily modified code
