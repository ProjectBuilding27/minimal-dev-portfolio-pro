## Full description

Minimal Developer Portfolio Pro is a clean, responsive and SEO-ready portfolio template for developers who want a professional online presence without spending days designing and coding from scratch.

Built with Next.js, React, TypeScript and Tailwind CSS, it includes a homepage, projects page, blog structure, uses page, contact page, sitemap, robots.txt and simple config-based customisation.

Edit one file, add your projects, deploy to Vercel and you're live.
